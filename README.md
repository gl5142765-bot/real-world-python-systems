# Real World Python Systems 🚀

This repository contains real-world backend system simulations built using Python.

## 📌 Projects

### 🎟️ Movie Ticket Booking System

* Seat selection system (A1, B2 format)
* Prevents double booking
* Cancellation & refund handling
* Ticket generation with logs

### 🚗 Parking Lot Management System

* Slot allocation & release
* Entry & exit tracking
* Prevents duplicate vehicle records
* Real-time slot availability

### 📦 QuickBuy Delivery System

* Order tracking system
* Delivery status updates
* Customer & order management

## ▶️ How to Run

  ## 📥 Setup

Download the project:
```
git clone https://github.com/gl5142765-bot/real-world-python-systems.git
```

Make sure Python is installed on your system.

Run any of the systems using the following commands in the Command system:
```

python movie_booking/cinebook.py
python parking_system/parking.py
python quickbuy_delivery/food_delivery.py
```
